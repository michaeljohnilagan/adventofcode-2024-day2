# Advent of Code 2024 Day 2 problem

Hello, world!

Here is my solution to the problem from [Day 2 of Advent of Code 2024](https://adventofcode.com/2024/day/2).
This is part of my application to the role of **Junior Machine Learning Engineer - Alerts Team** at Samdesk.

## Contents

This project repository contains two files.

* `sample.txt` is the sample puzzle provided in the problem description.
* `main.py` is the main Python script of my solution.

## How to use

To use this solution, execute the following, replacing `sample.txt` with your own puzzle data file.

```Bash
python3 -m main sample.txt
```

## Requirements

This project requires Python `>=3.10` and `numpy>=2.2.6`.

This project uses the `uv` package manager.
